# Tip Calculator Project

A modern Tip Calculator built with **Next.js**, **React Hook Form**, and **Zod** for form validation. Fully responsive, optimized, and designed with reusable components.

---

## Features

- **Bill, People, and Tip Inputs** with validation
- **Custom Tip Input** with real-time error messages
- **Preset Tip Buttons** (5%, 10%, 15%, 25%, 50%)
- **Dynamic Calculation** of tip per person and total per person
- **Reset Button** to clear all inputs
- **Enter Key Support**: pressing Enter blurs the input
- **Responsive UI** built with modern React components
- **Validation with Zod + React Hook Form**
  - Shows error for invalid inputs
  - Ignores empty fields until user types
